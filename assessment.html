Psychological Assessment

Assessment that clarifies complexity.

Psychological assessment remains part of the broader Stonebridge model, but it is not currently available for new requests. This page explains the intended direction of that work without implying current availability.

Not currently available

Assessment requests are not being accepted right now.

Stonebridge is not currently offering psychological assessment for new requests. The consultation form is configured to prevent assessment submissions so visitors are not misled into thinking this service is open.

The service remains visible because psychological assessment is part of the broader practice plan and may be developed later when the appropriate clinical, administrative, and documentation systems are in place.

Purpose

Assessment should answer a real question.

When offered in the future, assessment at Stonebridge should be used to clarify clinical complexity, not to produce jargon-heavy reports that obscure the person. The work should begin with a meaningful referral question and end with language that can be used.

01

Diagnostic and clinical clarification.

02

Careful integration of history, testing, and presentation.

03

Reports that translate findings into usable understanding.

Future Approach

Clear, careful, and clinically grounded.

Future assessment work should avoid both over-certainty and vague impressionistic language. It should hold complexity with enough discipline to be clinically useful and enough humility to remain accurate.

Clinical question first

Testing should be organized around the question being asked, not a generic battery.

Plain language

Findings should be readable and meaningful without losing professional precision.

Useful recommendations

Recommendations should follow from the data and the person’s actual context.

Boundaries

Not open for scheduling.

This page is informational. It does not indicate that assessment appointments are available. If you are seeking psychotherapy, use the consultation form. If you are seeking assessment specifically, Stonebridge is not currently the right referral point.

Current Availability

Psychotherapy is the current active service.

Return to the Services overview or submit a psychotherapy inquiry if that is the service you are seeking.
